# Transparent Taskbar

A simple Python script that makes the Windows taskbar fully transparent using the Win32 API.

## Author

Created by **died**

## Features

* Makes taskbar fully invisible
* Lightweight (no background apps)
* No third-party customization tools required

## Requirements

* Windows 10/11
* Python 3.x

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Notes

* This modifies the taskbar window directly using Windows API
* Transparency resets after restarting Windows Explorer or rebooting
